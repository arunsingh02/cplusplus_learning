# cplusplus_learning

## Create exe (build) file in c++
```
g++ -Wall -Wextra -Wpedantic -std=c++17 -o <exe-file-name> <cpp-file-name>
Ex.
  g++ -Wall -Wextra -Wpedantic -std=c++17 -o test test.cpp
```

## Run exe file
```
./<exe-file-name>
Ex.
  ./test
```
